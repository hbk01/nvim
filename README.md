# My (Neo)Vim configuration.

## Get started

### Dependency

+ NeoVim (or Vim)
+ nodejs (for coc.nvim plugin and more) (install neovim package)
+ python (install pynvim package)
+ python2 (optional if you installed python) (install pynvim package)
+ xclip (for clipboard function)
+ git (for clone github or gitee repositorys)
    + NOTE: if you are in China, github.com may not be accessible, please use proxy!
+ perl (optional) (install Neovim::Ext package)
+ ruby (optional) (install neovim package)

### NeoVim (recommend)

```shell
git clone https://github.com/hbk01/nvim --recursive ~/.config/nvim
```

### Vim

```shell
git clone https://github.com/hbk01/nvim --recursive ~/.vim
ln -s ~/.vim/init.vim ~/.vimrc
```
